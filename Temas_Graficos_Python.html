Estilos Disponíveis

Alguns dos estilos predefinidos mais populares incluem:

  • 'ggplot': Baseado no estilo do ggplot2 do R.
  • 'seaborn': Integra-se com a biblioteca Seaborn, oferece uma variedade de sub-estilos como 'seaborn-darkgrid', 'seaborn-whitegrid', etc.
  • 'bmh': Baseado no estilo do livro "Bayesian Methods for Hackers".
  • 'grayscale': Um estilo em tons de cinza.
  • 'fivethirtyeight': Baseado no estilo do site de jornalismo de dados FiveThirtyEight.

Exemplo de Código de implementação


                                import matplotlib.pyplot as plt
                                
                                # Aplicar um estilo específico
                                plt.style.use('seaborn-v0_8-poster')
                                
                                # Dados de exemplo
                                x = [1, 2, 3, 4, 5]
                                y = [2, 3, 5, 7, 11]
                                
                                # Criar o gráfico
                                plt.plot(x, y)
                                plt.title('seaborn-v0_8-poster')
                                plt.xlabel('Eixo X')
                                plt.ylabel('Eixo Y')
                                plt.show()
